Project Summary: Personality-Based Focus Timer

This Pomodoro Timer project was developed using **Entry** as a course assignment for **Information Engineering and Modern Society (CLC1260)**. The primary goal was to move beyond a basic time management tool and introduce a layer of **cognitive personalization** to truly enhance user focus and productivity.

The app's unique feature is its **personality-based music recommendation system**. It uses structured user input to suggest background music tracks that are specifically tailored to aid concentration based on common psychological profiles. This ensures the user is listening to music optimized for their individual focus state.
