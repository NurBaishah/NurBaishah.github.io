Project Focus: User-Centered Campus Solution

This project is currently in the **design and prototyping phase** for the **Human-Computer Interaction (HCI) (CAS2201)** course. Since this is an ongoing course project, there is no final coded product yet; the goal is the detailed design documentation and interactive prototype.

The core challenge involves designing a **seamless mobile experience** for Yonsei University students to efficiently report and recover lost items on a large campus. I am focusing on **intuitive UI/UX design** principles to simplify the complex workflows of item retrieval.

**Key Deliverables:** The process includes detailed user research, journey mapping (e.g., reporting a lost item vs. finding an item), creating wireframes, and building a high-fidelity, interactive prototype using **Figma**. This project demonstrates my skills in user flow creation and design thinking.
